# blog.co a blog ui with react

## Get Started

```bash
git clone https://github.com/arribion-link/blog.co.git
```

### Install all dependancies

- Get in backend directory
  
```bash
cd backend
```

```bash
npm install
```

- Get into frontend ( first cd ..)
  
```bash
cd frontend
```

```bash
npm install
```

#### Run the Project

```bash
npm run dev
```
